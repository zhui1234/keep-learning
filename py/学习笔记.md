### 1.5.常用命令

```bash
    conda creat -n 环境名称 python 指定版本  # 创建环境
    conda activate 环境名称  # 激活环境
    conda python --version  # 查看python版本
    conda env list  # 查看已创建环境
    conda install 需要安装的包名称(以空格分隔)  # 安装包
    conda list  # 查看已安装的包
    conda update 需要更新的包名称  # 更新包
    conda env export > 文件名.yaml  # 导出当前环境创建配置文件
    conda env creat -f 文件名.yaml -n 环境名称  # 从配置文件创造环境
    conda clean --all  # 清理conda缓存
```

## 2.Python学习

### 2.1.输入&输出

```python
    input("提示信息")
    # 输入的内容为字符串类型,需要用int(input()),float(input())等转换为指定类型数据
    print(object,sep,end)
    # object 要输出的一个或多个对象,用','分隔
    # sep 输出多个对象之间的分隔符,默认为' '
    # end 输出结尾的字符,默认为'\n'
```

### 2.2.循环

1. for

    ```python
    for i in range(5):  # 0,1,2,3,4

    for i in range(1,6):  # 1,2,3,4,5

    for i in range(1,10,2):  # 1,3,5,7,9

    test="Hello"
    for char in test:
        print(char,end=' ')  # 输出H e l l o  
    ```

2. while

    ```python
    while 条件:
        需要循环执行的代码
    else:
        循环正常结束执行的代码
    ```

3. 可使用`break`退出循环,`continue`跳过当前循环

### 2.3.字符串

1. 字符串长度

    ```python
    test="Hello World"
    length=len(test)  # length=11
    ```

2. 字符串访问和切片

    ```python
    test = "Hello World"

    print(test[0])  # H 第一个字符
    print(test[4])  # o 第五个字符
    print(test[-1]) # n 最后一个字符

    # [start:end:step]
    print(test[0:5])  # 
    print(test[6:])  #
    print(test[:5])  #
    print(test[::2])  # HloWrd
    print(test[::-1])  # dlroW olleH
    ```

### 3.2.图像数据

1. 基本元数据

    ```python
    img.shape # (高度, 宽度, 通道数)
    img.size # 总像素数
    img.dtype # 数据类型,通常是 uint8 (0-255)
    img.nbytes # 占用内存
    ```

2. 像素值数据

    ```text
    灰度图像(单通道),每个值代表点的亮度
    [[125, 130, 128, ...],
    [120, 135, 140, ...],
    [110, 115, 120, ...],
    ...]

    彩色图像(三通道),BGR图像则分别表示B,G,R的颜色值
    [[[255, 0, 0], [0, 255, 0], [0, 0, 255], ...],   # 蓝, 绿, 红
    [[128, 128, 128], [64, 64, 64], [192, 192, 192], ...],
    ...]
    ```

### 3.3.常用操作

1. 读取图片

    ```python
    ori_img=cv2.imread('文件名')
    ```

2. 打开/关闭图片

    ```python
    cv2.imshow('窗口名',ori_img)
    cv2.waitkey(0) #0:无限久  1000:1秒
    cv2.destroyAllWindows()
    ```

3. 处理图片

    ```python
    gray_img=cv2.cvtColor(ori_img,cv2.COLOR_BGR2GRAY)
    #cv2.cvtColor():转换图像的颜色空间
    #cv2.COLOR_BGR2GRAY:BGR彩色图像转为灰度图像
    blur_img=cv2.GaussianBlur(ori_img,(11,11),0)
    #cv2.GaussianBlur():应用高斯模糊
    #(11,11):卷积核大小(必须为奇数),越大越模糊
    #0:标准差(0表示自动计算),越大越模糊
    ```
